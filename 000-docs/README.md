# 000-docs

**Primary documentation directory for Bob's Brain.**

This directory contains all project documentation using the Document Filing System v2.0 naming convention:

**Format:** `NNN-CC-ABCD-description.md`
- **NNN** = Sequential number (001-999)
- **CC** = Category code (AA, PP, AT, etc.)
- **ABCD** = Document type (REPT, ARCH, etc.)
- **description** = 1-4 words, kebab-case

**Category Codes:**
- **AA** = After-Action Reports
- **PP** = Product & Planning
- **AT** = Architecture & Technical
- **TQ** = Testing & Quality
- **OD** = Operations & Deployment
- **LS** = Logs & Status
- **RA** = Reports & Analysis

Updated 2025-11-11.
