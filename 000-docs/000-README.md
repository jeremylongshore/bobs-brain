# 000-docs/

Top-level documentation for bobs-brain ADK agent.

## Document Filing System v2.0

All documents follow the naming convention: `NNN-CC-ABCD-description.md`

- **NNN** = Sequential number (001-999)
- **CC** = Category code
- **ABCD** = Document type (4-letter abbreviation)

### Category Codes

- **PP** = Product & Planning
- **AT** = Architecture & Technical
- **TQ** = Testing & Quality
- **OD** = Operations & Deployment
- **LS** = Logs & Status
- **RA** = Reports & Analysis
- **DR** = Documentation & Reference
- **AA** = After-Action Reports
